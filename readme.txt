Purpose: a personal website of links to my popular websites.
Features: 
* The entire "model" (data for the site) will be a single update-able csv file.
* The interface to the model should be a web api.
* Updates will be made through password only.
* The records of the CSV will be searchable by sub-string on all fields.
* The CSV records will be stored locally where local storage is supported.
* Responsive to multiple device sizes.
* Attempt at some primitive Agile development techniques.
* Server code in PHP with intention to switch to Ruby on Rails
* Best practices will be used during development:
    * YGANI
    * KISS
    * DRY
    * DOT
    * TDD
    * 7 LINES +/- 2
